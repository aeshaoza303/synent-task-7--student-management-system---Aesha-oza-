**🎓 Student Management System (CLI)**

A simple Python project that manages student records using a menu-driven interface and stores data in a JSON file.

**🚀 Features**

➕ Add new student records
📋 View all student details
🔍 Search student by roll number
✏️ Update student information
❌ Delete student records
💾 Stores data in JSON format

**🛠️ Technologies Used**
Python
JSON
OS
Regular Expressions (re)

**📂 Project Structure**

Student-Management-System/
│── students.json
│── README.md

**📊 Output**

A file named students.json will be created
It will contain:
Roll
Name
Age
Grade
Email
